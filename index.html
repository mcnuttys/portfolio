Unity Marching Cubes

Goal:

The goal was to create a fairly preformance friendly version of an infinate interactive terrain.

Description:

The project consists of a number of different implementations of marching cubes. In the "final" version the user is able to pretty much explore the terrain to their hearts content. The user is able to either paint terrain features or use the remove/add grenades. The reason I wanted to do this project was twofold, first I have always desired a game that had interactive terrain ala minecraft but not cube, and second to learn how to use marching cubes as I had previously tried like four times.

Interactive Meta Balls

Goal:

The goal was to create an interactive meta-balls projects, it is also supposed to be a tool to visualize the various steps in the meta-balls process

Description:

The Interactive Meta Balls project is a project made using JavaScript canvas. The user is able to change the direction the balls are moving, create new balls, and change a number of other things about the enviroment. The user is also able to change what is drawn onto the canvas, for example, they could change the drawing to only the anti-nodes and get a very different view of the meta-balls

Mirror Maze

Goal:

Mirror Maze is a game created for the 2019, GAME DEV club Game Jam at RIT. The theme of the jam was "behind you".

Description:

In the game you play as a charector who is placed at the center of a maze and are required to find 4 keys in order to exit. However, in order to navigate the player must use a mirror!

Game Info:

Controls:

  • WASD to walk around.
  • Move mouse to look around.
  • Mirror to see footprints, and other things.

Navigation:
The easiest way to navigate the maze is to simply stick to a wall, however, with what could be in the maze it is not always the safest option.
Warning, the sound is quite loud.
The game was the winner of the game jam.

Knight Light

Goal:

Knight Light is a game created for the 2019, Wild Magic Game Jam. The theme of the jam was "Tarot Cards".

Description:

In the game you play as a Knight who has a light and must clear rooms in an attempt to kill the boss at the end, and save their son. The player must navigate and find the enemies using only their meager light source.

Game Info:

Controls:

  • WASD to walk around.
  • Space to attack in the direction of previous and/or current movement.
  • Use light radius to see enemies.

Combat:
The easiest way to do combat is to simply take it slow, the controls can be a little hard to get used to.
Also, the sound is quite loud.
